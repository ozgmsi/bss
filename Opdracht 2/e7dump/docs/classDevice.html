Detailed Description

This class handles the interaction with the floppy and maintains the Block cache.

It is the equivalent of a "disk-driver".

Examples:
1.voorbeeld.cc, and 2.charter.cc.

void Device::setLimit ( daddr_x  dlimit)

Set the device limit on blocknumbers.

By default this "device driver" figures this out itself, but this method allows you to set it by hand. Setting it to 0 effectively sets it to infinite.

Parameters
[in]dlimitThe total number of blocks on this device.
Precondition
The new limit must be equal or less to what the driver already knows.
